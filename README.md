# CLI Log Parser for Tour Enquiries

## Project Description
This project is a Python-based CLI Log Parser that extracts customer names, email addresses, and travel destinations from a text file using Regular Expressions (Regex).

## Technologies Used
- Python
- Regular Expressions (Regex)

## Files
- parser.py
- enquiries.txt
- README.md
